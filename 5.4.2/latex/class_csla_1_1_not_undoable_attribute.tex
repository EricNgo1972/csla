\doxysubsection{Detailed Description}
Marks a field to indicate that the value should not be copied as part of the undo process. 

Marking a variable with this attribute will cause the n-\/level undo process to ignore that variable\textquotesingle{}s value. The value will not be included in a snapshot when Begin\+Edit is called, nor will it be restored when Cancel\+Edit is called.
