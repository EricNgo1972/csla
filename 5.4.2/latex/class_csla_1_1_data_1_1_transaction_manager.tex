\doxysubsection{Detailed Description}
Provides an automated way to reuse open database connections and associated ADO.\+NET transactions within the context of a single data portal operation. 


\begin{DoxyTemplParams}{Template Parameters}
{\em C} & Type of database connection object to use. \\
\hline
{\em T} & Type of ADO.\+NET transaction object to use. \\
\hline
\end{DoxyTemplParams}


This type stores the open ADO.\+NET transaction in Csla.\+Application\+Context.\+Local\+Context and uses reference counting through IDisposable to keep the transaction open for reuse by child objects, and to automatically dispose the transaction when the last consumer has called Dispose."{} \begin{Desc}
