\doxysubsection{Detailed Description}
This is the base class from which collections of editable root business objects should be derived. 


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Type of editable root object to contain within the collection. \\
\hline
\end{DoxyTemplParams}


Your subclass should implement a factory method and should override or overload Data\+Portal\+\_\+\+Fetch() to implement data retrieval. 

Saving (inserts or updates) of items in the collection should be handled through the \mbox{\hyperlink{class_csla_1_1_dynamic_list_base_a32e16f6c1f2df4595d6c1b7bbbdc253f}{Save\+Item()}} method on the collection. 

Removing an item from the collection through Remove() or Remove\+At() causes immediate deletion of the object, by calling the object\textquotesingle{}s Delete() and Save() methods. \begin{Desc}
