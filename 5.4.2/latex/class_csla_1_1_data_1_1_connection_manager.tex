\doxysubsection{Detailed Description}
Provides an automated way to reuse open database connections within the context of a single data portal operation. 

This type stores the open database connection in Csla.\+Application\+Context.\+Local\+Context and uses reference counting through IDisposable to keep the connection open for reuse by child objects, and to automatically dispose the connection when the last consumer has called Dispose."{}
