{\footnotesize\ttfamily Dictionary$<$string, \mbox{\hyperlink{class_csla_1_1_data_portal_client_1_1_data_portal_proxy_descriptor}{Data\+Portal\+Proxy\+Descriptor}}$>$ Csla.\+Data\+Portal\+Client.\+Data\+Portal\+Proxy\+Factory.\+Data\+Portal\+Type\+Proxy\+Descriptors\hspace{0.3cm}{\ttfamily [static]}, {\ttfamily [get]}}



Gets the list of proxy-\/type mapping descriptors used to create specific proxy objects for specific business classes. 

If a business type is not listed in this mapping then the default proxy and URL values from Application\+Context are used to create a proxy.

The key value is the first two elements of an assembly qualified type name (e.\+g. \textquotesingle{}System.\+String, mscorlib\textquotesingle{}).

Or the key value can be a resource id used to describe a business class via the Data\+Portal\+Server\+Resource attribute. (e.\+g. \textquotesingle{}Resource\+: 123\textquotesingle{}).
