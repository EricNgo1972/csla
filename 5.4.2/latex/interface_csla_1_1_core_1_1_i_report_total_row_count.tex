\doxysubsection{Detailed Description}
Implement this interface in a collection to report a total row count to \mbox{\hyperlink{class_csla_1_1_web_1_1_csla_data_source}{Csla.\+Web.\+Csla\+Data\+Source}}, where that row count is different from the collection\textquotesingle{}s normal Count property value. 

This interface is used to provide paging support for web data binding through \mbox{\hyperlink{class_csla_1_1_web_1_1_csla_data_source}{Csla.\+Web.\+Csla\+Data\+Source}}. You should implement this interface in your business collection class, along with windowed data loading, to provide efficient paging support.
